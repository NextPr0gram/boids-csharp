# Boids C#

A C# implementation of the Boids algorithm using SFML.Net.

This project was created to get the gist of the C# language and its syntax.

## What is Boids, and how does it work?

Boids is an algorithm that simulates the behavior of bird flocks and fish schools. It is based on three simple rules:

### 1. Cohesion
Each boid gradually steers toward its nearby neighbors.

### 2. Separation
Each boid steers away from others when they get too close.

### 3. Alignment
Each boid tries to match the speed and direction of nearby boids.
